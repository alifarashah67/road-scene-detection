# Road Scene Detection System

A practical computer vision project for real-time road scene object detection using YOLOv8 and OpenCV.

## Features
- Upload road images
- Detect vehicles, pedestrians, and scene objects
- Visualize detection results
- Lightweight demo with Streamlit

## Tech Stack
- Python
- YOLOv8
- OpenCV
- Streamlit

## Use Case
This project is designed as a perception demo for autonomous mobility and intelligent transportation applications.
